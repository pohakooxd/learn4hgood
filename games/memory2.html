Playing Cards Memory

Playing Cards Memory is a fun, online, card flipping memory game where you must memorize where certain cards are and then match them up to make money! Each flip of a card costs money and you only have a certain amount to start off with so make it last. As if that wasn't tough enough, you will also be against the clock. The timer will be ticking down so everything will be against you. Have you got enough brain power to take on the challenge, and come out victorious?

The aim of the game is simple. You must match sets of two identical cards. All cards are face down so you can only see what card they are by flicking them over, which costs money. The cards will not shuffle around so keep in mind which card is where if you make an unsuccessful match. This game will test and develop your memory skills. Work methodically so you are not wasting money flipping over random cards all the time. Come up with a clever process to minimize the amount of flips you need to make.
